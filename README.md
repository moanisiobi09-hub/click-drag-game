This is the most substantial project I have on GitHUB (at the time of its upload). It's another fairly simple game created using the visual
framework SFML for C++. I handcoded all the click-and-drag calculations [i.e. those needed to move the circular 'knobs' around the central
'gamesquare', keeping the knobs moving in a circle while moving naturally to imprecise player-inputted dragging] from scratch.
